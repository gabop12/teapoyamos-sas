Análisis Frameworks

Análisis del Problema

La firma “Te Apoyamos SAS” busca ampliar su cobertura de servicios a través de una aplicación web. Actualmente enfrenta la ausencia de un sistema que permita centralizar la información institucional, gestionar hojas de vida de aspirantes y registrar procesos legales de clientes.

  • Requerimientos funcionales: portal informativo, registro de aspirantes, gestión de procesos legales, módulo de noticias y formulario de contacto.
  • Requerimientos no funcionales: seguridad de datos, disponibilidad, usabilidad, rendimiento y cumplimiento normativo.
  • Actores involucrados: administrador, directivos, profesionales adscritos, aspirantes, clientes y público en general.
  • Información a registrar: datos institucionales, profesionales, aspirantes, clientes, procesos legales, noticias y contactos externos.

Investigación sobre Frameworks

Para el desarrollo de páginas web es importante apoyarse en frameworks que permitan ahorrar tiempo y lograr que el sitio se adapte a cualquier pantalla. Entre los más relevantes se encuentran:

  • Bootstrap: Caja de herramientas con plantillas y componentes listos. Facilita diseño responsive.
  • Tailwind CSS: Framework moderno para personalizar estilos de forma rápida y única.
  • WordPress: CMS que permite crear y administrar sitios web completos sin conocimientos avanzados.
  • React: Herramienta avanzada para crear aplicaciones web interactivas y rápidas.
